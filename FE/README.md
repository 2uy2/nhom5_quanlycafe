# Smart Cafe Operations UI

Giao diện ReactJS cho use case PB01 - Quản lý menu trong hệ thống vận hành quán cà phê thông minh.

## Chạy frontend

```bash
npm install
npm run dev
```

## Chức năng đã có

- Hiển thị danh sách món: tên món, giá, mô tả, hình ảnh, trạng thái, thời gian cập nhật.
- Thêm mới, cập nhật, xóa và xem chi tiết món.
- Kiểm tra dữ liệu đầu vào cho tên món, giá, mô tả và URL hình ảnh.
- Thông báo kết quả thao tác cho admin.
- Lưu dữ liệu demo bằng `localStorage` để giao diện hoạt động độc lập, không cần backend.
